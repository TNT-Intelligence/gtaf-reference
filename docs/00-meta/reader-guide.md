# Reader Guide

## Normative language
- **MUST** = required for validity
- **SHOULD** = recommended baseline
- **MAY** = optional if explicitly defined
- **NON-BINDING** = illustrative only; not normative; does not create requirements

Note: GTAF defines structural prerequisites for delegation; it does not guarantee decision correctness.
